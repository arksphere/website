## Introduction

Archon is the command center for AI coding assistants, integrating knowledge management, smart search, and task collaboration. It supports multiple MCP protocol AI assistants.

## Key Features

- Project knowledge base and documentation management
- Smart semantic search with advanced RAG strategies
- Task management and real-time collaboration
- Connects to various AI coding assistants

## Use Cases

- Team and personal project knowledge management
- Context sharing for AI coding assistants
- Intelligent retrieval of code, docs, and tasks
- Real-time collaboration and task assignment

## Technical Highlights

- MCP protocol server, compatible with Claude Code, Cursor, etc.
- Supports website crawling, PDF/doc upload
- Real-time content updates and collaboration
- Open-source architecture, easy to extend
