[agenticSeek](https://github.com/Fosowl/agenticSeek) is an open-source, local agentic search and AI agent platform by the Fosowl team. It supports multiple AI Agent scenarios, suitable for agent retrieval, knowledge discovery, and more.

agenticSeek is a 100% local alternative to Manus AI, featuring voice control, autonomous web browsing, code writing, and task planning. All data remains on your local device, ensuring privacy and security with no cloud dependency.

## Why Choose agenticSeek?

- **Fully Local & Privacy Protection**: All features run locally—no cloud, no data sharing. Files, conversations, and searches are private.
- **Intelligent Web Browsing**: Autonomously searches, reads, extracts information, and fills web forms for fully automated operations.
- **Autonomous Coding Assistant**: Supports code writing, debugging, and running in Python, C, Go, Java, and more—no manual intervention required.
- **Automatic Agent Selection**: Automatically picks the best agent for each task, like having an expert team on standby.
- **Complex Task Planning & Execution**: Breaks down large tasks into steps, coordinates multiple AI Agents for collaboration—ideal for travel planning, project management, etc.
- **Voice Control**: Supports voice wake-up and speech-to-text for a sci-fi personal AI assistant experience (feature in development).

## Use Cases

- Local AI Agent solutions
- RAG retrieval and knowledge base
- Automated web operations
- Personal privacy protection
- Complex task automation
