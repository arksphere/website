## Introduction

Langfuse is an open-source platform for LLM application development, supporting team collaboration, monitoring, evaluation, and debugging. Users can self-host or use the cloud service, quickly integrate mainstream LLM frameworks, and enhance observability and development efficiency for AI applications.

## Key Features

- LLM call tracing and log analysis
- Prompt management and version control
- Multi-dimensional evaluation and dataset management
- Rich API and SDK integrations
- Visual playground and real-time debugging

## Use Cases

- Enterprise-level LLM application development and operations
- Multi-team AI project management
- Monitoring and optimization of complex AI workflows
- Evaluation and continuous improvement of LLM products

## Technical Highlights

- Supports self-hosting and cloud deployment, easy integration
- Open-source architecture, compatible with mainstream LLM frameworks
- Comprehensive API and OpenTelemetry integration
- High-performance data collection and analysis
