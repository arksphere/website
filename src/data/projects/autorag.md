## Introduction

AutoRAG is a toolkit for rapidly building RAG pipelines, automating document indexing, retrieval, and integration with generative models.

## Key Features

- Automated document processing and vector index construction.
- Compatible with multiple generative models and vector databases.

## Use Cases

- Rapid prototyping and production deployment of Retrieval-Augmented Generation (RAG) systems.

## Technical Highlights

- Provides configurable indexing and retrieval strategies to suit different data scales and latency requirements.
