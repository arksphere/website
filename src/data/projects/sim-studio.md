Sim Studio is a lightweight and user-friendly platform for building AI agent workflows, offering both cloud-hosted and self-hosted deployment options with extensive technical capabilities.

## Platform Overview

Sim Studio simplifies AI agent workflow construction through an intuitive interface and robust backend support, enabling developers to quickly create, deploy, and manage complex AI workflows.

## Key Features

- **Visual Workflow Editor**: Drag-and-drop interface for workflow design
- **Multiple Deployment Options**: Cloud-hosted and self-hosted solutions
- **AI Agent Management**: Support for multiple AI agents with role-based capabilities
- **Real-time Monitoring**: Comprehensive workflow monitoring and debugging tools

## Technical Stack

- Frontend: Next.js, Shadcn, Tailwind CSS, ReactFlow
- Backend: Bun, PostgreSQL, Drizzle ORM
- State Management: Zustand, Socket.io
- Development Tools: TypeScript, ESLint, Prettier
