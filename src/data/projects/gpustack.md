## Introduction

gpustack provides GPU cluster resource management and job scheduling tools, helping teams efficiently manage training and inference workloads using GPU resources.

## Key Features

- Cluster monitoring and resource scheduling.
- Job queue and priority management.

## Use Cases

- GPU resource management and training scheduling in research or production environments.

## Technical Highlights

- Integrates with common container and scheduling systems, supports extensible plugins.
